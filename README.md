# Hub Numérique Accessible

Page de présentation des projets numériques accessibles de Saif / Atelier Numérique 37.

## Objectif

Ce dépôt contient une page statique destinée à être publiée avec GitHub Pages.

Elle présente les projets suivants :

- H2VL Connect
- Audio Memo
- Petits Chevaux
- Transport Tours
- Localisateur MCP

## Déploiement GitHub Pages

Dans GitHub :

1. Ouvrir les paramètres du dépôt.
2. Aller dans **Pages**.
3. Choisir la branche `main`.
4. Choisir le dossier `/root`.
5. Enregistrer.

L’URL devrait être :

```text
https://ateliernumerique37-tech.github.io/hub-numerique/
```

## Structure

```text
index.html
style.css
.nojekyll
README.md
```

## Choix techniques

- HTML et CSS simples.
- Aucun framework.
- Aucun build.
- Compatible GitHub Pages.
- Structure pensée pour l’accessibilité : titres hiérarchiques, lien d’évitement, navigation claire, contrastes, focus visible, prise en compte de `prefers-reduced-motion`.
