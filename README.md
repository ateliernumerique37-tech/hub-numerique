# Hub Numérique Accessible

Page professionnelle de présentation des projets numériques accessibles de Saif Ayedi / Atelier Numérique 37.

## Site publié

```text
https://ateliernumerique37-tech.github.io/hub-numerique/
```

## Objectif

Ce dépôt contient une page statique publiée avec GitHub Pages.

Elle présente un écosystème de projets autour de l’accessibilité numérique, des usages associatifs, des jeux accessibles et des services utiles au quotidien.

## Projets présentés

- H2VL Connect : application interne de gestion associative, accès privé.
- Audio Memo : jeu de mémoire sonore accessible.
- Petits Chevaux : jeu accessible en PWA, public.
- Transport Tours : skill Alexa et API transport, démonstration sur demande.
- Localisateur MCP : serveur MCP connecté à localisateur.org.
- Apprenti Clavier : projet d’apprentissage du clavier avec lecteurs d’écran.

## Déploiement GitHub Pages

Dans GitHub :

1. Ouvrir les paramètres du dépôt.
2. Aller dans **Pages**.
3. Choisir la branche `main`.
4. Choisir le dossier `/root`.
5. Enregistrer.

## Structure

```text
index.html
style.css
.nojekyll
README.md
```

## Choix techniques

- HTML et CSS simples.
- Aucun framework.
- Aucun build.
- Compatible GitHub Pages.
- Structure pensée pour l’accessibilité : titres hiérarchiques, lien d’évitement, navigation claire, contrastes, focus visible et mode sombre automatique.
