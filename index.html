Pourquoi ce hub ?

Ce hub rassemble des outils créés ou pilotés par Saif autour d’un même objectif : rendre le numérique plus utile, plus accessible et plus humain, notamment pour les personnes déficientes visuelles et les associations locales.

Les projets présentés ici couvrent la gestion associative, les jeux accessibles, les transports à Tours et les outils de recherche adaptés aux usages avec lecteur d’écran ou assistant vocal.

Les projets

H2VL Connect

Application interne de gestion associative pour H2VL : adhérents, événements, inscriptions, cotisations, communications e-mail, rôles administrateurs et tâches d’équipe.

Gestion associative accessible et structurée.

Audio Memo

Jeu de mémoire sonore en PWA, conçu en priorité pour les personnes non-voyantes et malvoyantes, avec une expérience pilotable au lecteur d’écran.

Jouer avec les sons, pas avec les yeux.

Petits Chevaux

Version accessible du jeu des Petits Chevaux, jouable sur mobile et ordinateur, avec mode local, IA Bernard et multijoueur en ligne.

Un classique remis sur les rails de l’accessibilité.

Transport Tours

Skill Alexa et API permettant de demander vocalement les prochains passages du réseau Fil Bleu à Tours, avec une réponse courte, claire et utile.

Demander son bus ou son tram, simplement.

Localisateur MCP

Serveur MCP pour interagir avec localisateur.org depuis Claude : annuaires, dictionnaires, radios, météo, transports, recettes, vie pratique et autres services utiles.

Un portail accessible augmenté par l’IA.

L’approche

Accessibilité d’abord

Les interfaces sont pensées pour les lecteurs d’écran, la navigation clavier, les annonces vocales et les usages sans repère visuel.

Utilité concrète

Chaque projet répond à un besoin réel : organiser une association, jouer, se déplacer, chercher une information ou préparer un atelier numérique.

Maintenance raisonnable

Les choix techniques privilégient la robustesse : PWA, Firebase, GitHub Pages, VPS léger, documentation Markdown et architecture compréhensible.

Contact

Ce hub est en construction. Il servira de vitrine pour présenter les projets, documenter leur usage et partager les avancées autour du numérique accessible.

Pour une démonstration, une question ou une collaboration, contacte Saif via les canaux habituels de l’Atelier Numérique 37.
